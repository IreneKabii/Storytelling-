📊 Visual-Narrative: The Art of Data Storytelling
📌 Project Overview
Data is only as valuable as the story it tells. In this project, I utilize R and the Grammar of Graphics (ggplot2) to transform abstract numbers into intuitive, visual narratives. By focusing on [Insert Topic, e.g., Kenyan Market Trends], I demonstrate how high-fidelity visuals can uncover hidden correlations and seasonal patterns that traditional spreadsheets overlook.
🎯 Objectives
• Information Density: Reducing cognitive load by presenting complex multi-variable data in clear, faceted visualizations.
• Aesthetic Precision: Using custom themes and professional color palettes to ensure charts are "presentation-ready" for executive stakeholders.
• Actionable Insight: Moving beyond "what" the data is to "why" it matters for business strategy.
🛠 Tech Stack
• Language: R
• Visualization: ggplot2, patchwork (for plot composition), ggthemes.
• Data Wrangling: tidyverse, scales.
• Reporting: R Markdown & Quarto.
📂 Repository Contents
• scripts/: Modular R scripts containing custom visualization functions.
• visuals/: A gallery of high-resolution .png exports of the key findings.
• analysis.Rmd: A reproducible report linking the code to the narrative conclusions.
